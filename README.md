ATM Simulation System

A simple ATM Simulation System developed in Python. This command-line application simulates basic ATM operations such as balance inquiry, cash deposit, and cash withdrawal after successful PIN authentication.

Project Overview

The purpose of this project is to demonstrate the implementation of basic banking operations using Python. It is designed for beginners to practice fundamental programming concepts, including conditional statements, user input, variables, and arithmetic operations.


 Features

- PIN Authentication
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Insufficient Balance Validation
- Menu-Driven Interface

 Technologies Used

- Python 3
- Visual Studio Code

 Project Structure

 Requirements

Installation

Clone the repository:

```bash
git clone https://github.com/your-username/atm-simulation-system.git
```

Navigate to the project directory:

```bash
cd atm-simulation-system
```

Run the program:

```bash
python atm.py
```
